SqlAgent 项目介绍
Created2026-04-17|项目文档
1. 项目概述 SqlAgent 是一个面向自然语言数据分析场景的 AI Agent 系统，目标是把“提问 → 检索 → 生成 SQL → 执行 → 修复 → 分析/可视化”串成可观测、可扩展、可复用的闭环。 项目已经整体完成，系统能力覆盖多智能体协作、分层 Schema + Value Lens 增强、MCP 风格协议封装与 Prompt 自动优化机制，能够稳定支持从业务问题到数据结论的端到端自动化分析。 2. 背景与价值 🎯 为什么要做这个项目 企业数据分析普遍存在 SQL 编写门槛高、问题语义理解成本高、跨表分析慢等问题。 🛠️ 解决什么问题 将业务问题自动转化为可执行 SQL，并在失败时自动修复，同时输出结构化结果与可读分析报告。 降低“人工写 SQL、人工排错、人工解释”的时间成本。 👥 适用对象 业务分析师、数据产品经理、数据工程团队、需要快速验证经营问题的运营团队。 3. 项目目标 🚀 3.1 功能目标🛡️ 3.2 非功能目标 目标 A：支持自然语言到 SQL 的端到端生成与执行。...
